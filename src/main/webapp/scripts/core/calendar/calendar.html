Calendar

Description

The Spry Calendar is a UI widget that displays a dynamic calendar with forward and reverse navigation as well as clickable dates for date selection.

Required Files

SpryCalendar.js

SpryCalendar.css

Reference File

SpryCalendar.html

Sample Files

SpryCalendarSample.html

Constructor

Widget Constructors are small pieces of javascript that activate the markup into the working widget. These scripts must come AFTER the markup on the page, since the markup needs to exist before the constructor fires.

<script language="javascript" type="text/javascript">
	var cal1 = new Spry.Widget.Calendar("cal1");
</script>

Basic Constructor

A basic constructor specifies the name of the widget and identifies the ID of the main markup container. The name of the widget is used to identify the widget for functions and methods.

<script language="javascript" type="text/javascript">
	var widgetname = new Spry.Widget.Calendar("id of widget table");
</script>

Constructor Options

Constructor options allow users to specify certain attributes of the widget.

Constructor options follow the ID parameter, wrapped in curly braces {}. Options are name:value pairs, separated by a colon (:).

<script language="javascript" type="text/javascript">
	var widgetname = new Spry.Widget.Calendar("id of widget table", {option1:value, option2:value, option3:"value"});
</script>
Option Values Default Description
year number Current year from a new Date object Default calendar year
month number (0-11) Current month from a new Date object Default calendar month
calendarMonths array ["January","February","March", "April","May","June", "July","August","September", "October","November","December"] Name of the months used in the calendar header.
headerRowId string null Table row CSS identifier for the row to be used as the header. If null, the first row of the table will be used.
enableKeyboardNavigation true or false true Allow the user to use the keyboard for navigation between days of the calendar.
previousDayKeyCode number 37 (left arrow key) Key code to allow users to navigate the focused day to the previous day using the keyboard.
nextDayKeyCode number 39 (right arrow key) Key code to allow users to navigate the focused day to the next day using the keyboard.
spaceKeyCode number 32 (space key) Key code to allow users to set the selected date using the keyboard.
selectedDate Date null Currently selected date.
callbackFunction function null Function that will be assigned to handle the date clicks. This function must accept a Date object as a parameter.
calendarClass CSS class name Calendar Style the table.
foreignDayClass CSS class name CalendarForeignDay Style the table cells that do not fall within the current calendar month.
monthHeaderClass CSS class name CalendarMonthHeader Style the header cells
navOverClass CSS class name CalendarNavOver Style the header navigation cells when a user mouses over them.
dayHeaderClass CSS class name CalendarDayHeader Style the day header cells.
dayClass CSS class name CalendarDayCell Default style for the day cells. Each cell that is not in the header rows MUST have this class added to the HTML markup at design time.
dayOverClass CSS class name CalendarDayCellOver Style the day cells when a user mouses over them.
todayClass CSS class name CalendarTodayCell Highlight today's date if it is in the current calendar month.
selectedClass CSS class name CalendarSelectedCell Highlight the selected date if it is in the current calendar month.
focusedCellClass CSS class name CalendarFocusedDay Highlight the day cell that currently has focus.
focusedClass CSS class name CalendarFocused Highlight the calendar element when it has focus. 
<script language="javascript" type="text/javascript">
	// Used as the calendar callback function.
	var cb = function(val)
	{
		alert("You clicked on " + val + ".");
	}
	// The date we want selected in the calendar.
	var sd = new Date("1/1/2009");
	
	var cal1 = new Spry.Widget.Calendar("cal1",{month:4, year:2010, callbackFunction:cb, selectedDate:sd});
</script>

The above example creates a Spry calendar from the table with an ID of "cal1" with May 2010 as the default month. When a date is clicked, the callback function triggers an alert box telling the user which date he/she clicked. This also changes the selected date from January 1, 2009 to the date that was clicked.

Remember that JavaScript is case sensitive and a Date object's months are zero-based.

 

Calendar Methods

clearSelection

Clears the selected date and removes the highlight style from the table cell if it is showing.

Format

Calendar.clearSelection()

Example

<a href="javascript:void(0)" onclick="cal1.clearSelection();">Clear selection</a>

getSelectedDate

Returns the selectedDate Date object.

Format

Calendar.getSelectedDate()

Example

<script language="javascript" type="text/javascript">
	function viewNextMonth()
	{
		var nm = cal1.getSelectedDate().getUTCMonth() + 1;
		var ny = cal1.getSelectedDate().getUTCFullYear();
		cal1.setCalendarDate(ny, nm);
		return false;
	}
</script>

<a href="javascript:void(0)" onclick="viewNextMonth();">View next month</a>

setCalendarDate

Redraws the calendar with a new month and year.

Format

Calendar.setCalendarDate(number yr, number mo)

Example

<a href="javascript:void(0)" onclick="cal1.setCalendarDate(2010, 4);">Set date to May 2010</a>

setSelectedDate

Allows setting of the selected date for the calendar so the day will be highlighted when displayed using script code rather than having to click on a date cell.

Format

Calendar.setSelectedDate(mixed val)

Example

<a href="javascript:void(0)" onclick="cal1.setSelectedDate("1/1/2009");">Set selected date to January 1, 2010.</a>

<a href="javascript:void(0)" onclick="cal1.setSelectedDate(1262325600000);">Set selected date to January 1, 2010.</a>
